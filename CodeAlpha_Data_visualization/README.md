# CodeAlpha - Data Visualization

## Project Description
This project is completed as part of the CodeAlpha Data Analytics Internship.

## Dataset
Sample Superstore Dataset

## Tools Used
- Python
- Pandas

- Seaborn
- Jupyter Notebook (VS Code)

- Sales Distribution

- Sales Share by Category
- Profit Distribution
